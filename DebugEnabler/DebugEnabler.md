# Debug Enabler

A BepInEx plugin that restores and enables the developer debug menu, and automatically bypasses the debug password prompt.  

**Use the `Del` key to open the menu.**

---

### Notable Features
- Item Spawner
- Give Gold
- Fortune/Status Giver
- Unlock All Roguelike Characters
- Unlock All Roguelike Difficulties
- Remove Summon Limit
- Kill All/Closest/Enemies
- Kill All/Selected Player
- Level Up Characters *(consider using my other mod for this)*

### Installation
1. Clone or download this repository.  
2. Build the project (if needed).  
3. Place `Debug Enabler.dll` into your `Stolen Realm\BepInEx\plugins` folder.  

---

### Notes
- Compatible with the latest version of Stolen Realm.  
- Use responsibly; enabling debug features may affect gameplay.
